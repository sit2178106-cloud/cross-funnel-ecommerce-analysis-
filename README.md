# Prada 2025: Synthetic Cross-Funnel E-Commerce Analysis

This repository contains the Python scripts and datasets used for my thesis on **"Cross-Funnel E-Commerce Analytics in the Luxury Fashion Industry."

# Project Overview
Due to data privacy regulations in the luxury sector, a robust, statistically sound synthetic dataset was generated mirroring real-world e-commerce KPIs (Prada 2025 projections). The analysis connects Top-Funnel (Marketing Campaigns), Mid-Funnel (Web Analytics), and Bottom-Funnel (CRM & Profitability) data.

# Repository Structure
* `thesis_dataset.py`: The script used to generate a relational database architecture with normal and Pareto distributions.
* `thesis analyze.py`: The final analysis script utilizing Fractional Attribution, Logistic Regression, ANOVA, and Cross-Funnel Pearson Correlations.
  
* datasets/`: Contains the Top, Mid, and Bottom funnel CSV files.
